## Changes Made

### Wave 1: XR Controller Overhaul + Course Theming
**Files modified:** xrinput.ts, ui.ts, game.ts, audio.ts, effects.ts, hud.ts, environment.ts, browserinput.ts, index.ts, controls.ts

1. **xrinput.ts** — Complete rewrite (86→270 lines):
   - State-based input routing for ALL game states
   - Trigger to putt/select, A to confirm/skip, B to reset/back
   - Grip button for power boost multiplier
   - Left thumbstick camera orbit → shared with browserinput
   - Right thumbstick menu navigation with cooldown
   - Laser pointer visualization (line + endpoint dot)
   - Focus highlight system for menu items

2. **ui.ts** — Major overhaul (330→420 lines):
   - `handleNavNext()`/`handleNavPrev()`/`handleBack()` for XR thumbstick
   - Focus index tracking with visual outline+glow
   - Course-specific color themes throughout
   - Practice mode button on each course card
   - Mini result table in course complete screen
   - VR control hints on screens

3. **game.ts** — Enhanced (298→330 lines):
   - 10-stroke limit per hole (`MAX_STROKES_PER_HOLE`)
   - `skipToNextHole()` for instant advance
   - `startPracticeHole()` for practice mode
   - `practiceMode` flag (replays same hole)
   - `courseManager` and `audio` made accessible

4. **environment.ts** — Rewritten (213→320 lines):
   - 3 distinct course themes with unique fog, grid, starfield, geometry, lights
   - `applyTheme()` dynamically swaps environment
   - Mixed geometry shapes (box, sphere, torus, cone)

5. **audio.ts** — Enhanced (293→310 lines):
   - `switchCourseAmbient()` with 3 distinct drone configs
   - Course-specific base frequencies, LFO rates, pad types
   - Sub-bass for courses 1 & 2
   - `playStrokeLimit()` descending "wah-wah" sound

6. **effects.ts** — Enhanced (207→260 lines):
   - Course-specific particle palettes
   - Pulsing orb celebration effects
   - `strokeLimitEffect()` red/orange failure particles
   - Memory cleanup: dispose geometry/materials

7. **hud.ts** — Enhanced (265→280 lines):
   - Stroke limit indicator with color warning
   - Course-themed border and accent colors

### Wave 2: Mini-Map + Practice Mode
**New files:** minimap.ts (175 lines), practice.ts (155 lines)

8. **minimap.ts** — New:
   - Canvas-based overhead 2D mini-map
   - Shows hole surfaces, walls, obstacles
   - Ball position (cyan with glow), tee (green), cup (pulsing yellow)
   - Course-themed border color
   - Performance-optimized (renders every 3rd frame)

9. **practice.ts** — New:
   - Hole selection grid for any course
   - Best practice scores saved in localStorage
   - Integrates with game manager's practice mode

10. **README.md** — Comprehensive documentation
